# Yenten GUI CPU miner (multi coins)
*GUI miner for CPU coins*

Supported algorithms:
 - yespower
 - yespowerr16
 
 Supported coins:
  - Yenten
  - Cranepay
  
  Supported pools:
   - cpu-pool
   - rplant pool
   - p2pool
   - bpool
   - antminepool
